HTML

HTML, o HyperText Markup Language, es el lenguaje base que se usa para crear páginas web. Funciona como los cimientos y la estructura de un edificio, definiendo cómo se organizan y muestran los diferentes elementos en una página.

Estructura

Etiquetas: Son los bloques de contruccion algunos ejemplos son el div, section, etc.
Atributos: Detallan mas la informacion de las etiquetas como class,id, src.
Estructura: Usa etiquetas como header, nav, footer.
Enlaces: Crea enlaces con a, e inserta imagenes con img.

Resumen

HTML funciona como un esqueleto para las paginas web y funciona a base de etiquetas que indican como mostrar el contenido

CSS

Las hojas de estilo en cascada o CSS (Cascading Style Sheets) es un lenguaje que define la apariencia de un documento escrito en un lenguaje de marcado como HTML Son un mecanismo simple para agregar estilo o diseño por ejemplo, fuentes, colores, espaciado a documentos web.

Apuntes

Hay tres formas de poner estilo de CSS en el HTML:

  • Hojas de estilo incrustada: Una hoja de estilo puede incrustarse en un documento con la etiqueta Una hoja de estilo incrustada puede usarse en un documento único. Si la misma hoja de estilo se usa para múltiples documentos, entonces seria mas apropiada una hoja de estilo externa.
  • Hoja de estilo en línea: Puede usarse estilos en linea mediante al atributo style El atributo style puede ser aplicado a cualquier elemento HTML.
  • Hoja de estilo externa: Una hoja de estilo externa puede ser anlazada a un documento HTML mediante el elemento link de HTML
  • Cada forma funciona de manera diferente y tiene ventajas y desventajas

Resumen

El CSS se utiliza para darle toda la imagen a la pagina web,

Etiqueta a
La etiqueta a en HTML, conocida como "anchor" o ancla, se usa para crear enlaces a otras páginas web o a diferentes partes de la misma página.

Apuntes

Href es un atributo que especifica la URL a la que el enlace debe llevar. Puedes también hacer enlaces dentro de la misma página usando identificadores

  • Href
    • Es un atributo de la etiqueta a que define la URL a la que se redirige cuando se hace clic en el enlace

  • Id
    • El id es un identificador unico para un elemento HTML. Se usa para estilos CSS especificos o para referenciar con enlaces internos.

    Resumen

    Esta es una caracteristica esencial de HTML, un recurso necesario para mejorar la organizacion de la pagina y para incluir recursos externos.

    Links de imagen
    Podemos usar una imagen como link, simplemente poniendo < img > dentro de la etiqueta < a > Ejemplo: < a href=”#” >< img scr=”imagen.jpg” >< /a >

    Apuntes

    Links con atributo target El atributo target especifica donde abrir el documento vinculado y puede tener los siguientes valores:

  • _self:predeterminado. Abre el documento en la misma ventana /pestaña en la que se hio clic
  • _blank. Abre el documento en una nueva ventana o pestaña
  • _parent. Abre el documento en el marco primario
  • _top. Abre el documento en el cuerpo completo de la ventana

    Resumen

    El texto se compone de encabezados y parrafos

    En HTML hay seis elementos de encabezados

  • < h1 >, < h2 >, < h3 >… < h6 >
  • Cada elemento representa un nivel de contenido diferente en el documento;
  • < h1 > Representa el titulo principal
  • < h2 > Representa el subtitulo
  • < h3 > Subtitulo de subtitulo, etc Párrafos
  • En HTML cada párrafo tiene que estar delimitado por un elemento
  • Listas
    • En HTML hay diferentes tipos de listas. Listas no ordenadas se usan para marcar listas cuyo orden no importa
